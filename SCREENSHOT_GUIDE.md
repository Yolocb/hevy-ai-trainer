## üõ†Ô∏è How to Capture Screenshots

### Option 1: Windows Snipping Tool (Recommended)
1. Press `Win + Shift + S`
2. Select area to capture
3. Screenshot copies to clipboard
4. Open Paint ‚Üí Paste ‚Üí Save as PNG

### Option 2: Browser DevTools
1. Press `F12` in browser
2. Press `Ctrl + Shift + P`
3. Type "screenshot"
4. Select "Capture full size screenshot"

### Option 3: ShareX (Best Quality)
1. Download ShareX (free)
2. Press `Ctrl + Print Screen`
3. Select region
4. Auto-saves to Pictures folder
