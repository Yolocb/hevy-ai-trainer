## Translation System

### How It Works

1. **Backend Generation:** The server generates routines with LOTR-themed names
2. **Frontend Translation:** JavaScript automatically translates names based on active theme
3. **Real-time Switching:** Toggle themes to see names instantly transform
4. **Fallback System:** Any unmapped names are auto-converted (uppercase + symbol replacement)
